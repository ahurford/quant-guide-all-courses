Chapter 9 BIOL 1001: LABORATORY 2

BIOLOGICAL DATA WITH COMPUTERS

PURPOSE

  1. To become familiarize with R Studio
  2. To learn how to record data in electronic format
  3. To learn how to write hypotheses as equations
  4. To learn how to choose the appropriate visualizations
  5. To learn how to make graphs using R Studio

Before coming to the laboratory:

  1. if you wish to use your own labtop, then install R and RStudio. For instructions see here. insert link

  1. Intro to R
  2. Entering data exercise. Identify relevant data. Data should be discrete with at least 3 groups. Export from R as .csv, write metadata. Submit to dropbox.
  3. Plot data to replicate a bar chart. Replot as box plot. Save in .pdf or related format.
  4. Practice writing hypothesis, prediction, and equation (i.e., equation is means of groups not equal in notation).
  5. Repeat plotting for data with continuous independent variable.

  6. Practice writing equation for data with continuous independent variable.

  7. Repeat plotting for data with continuous independent variable.

  8. Practice writing equation for data with continuous independent variable.

  1. In the bar graph, what can you tell about the stomatal openings of the treatment plant vs. the control plants? Is it consistent across all the individual plants?
  2. Label the following on the box plot (mean, 25% and 75% quantiles, 95% confidence limits, outliers).
